# 🚀 Task Manager Web Application

A modern, responsive **Task Manager (To-Do App)** built using **HTML, CSS, and JavaScript**.
This project demonstrates clean UI design, state management, browser storage handling, and performance optimization.

🔗 **Live Demo:**
https://omkarwork24.github.io/To-Do-Website/

💻 **GitHub Repository:**
https://github.com/omkarwork24/To-Do-Website

---

## 📌 Features

* ✅ Add new tasks
* ✅ Mark tasks as completed
* ✅ Delete tasks
* ✅ Real-time search
* ✅ Filter (All / Active / Completed)
* ✅ Progress tracking bar
* ✅ Confetti celebration on completion
* ✅ Background slideshow (optimized)
* ✅ Dark mode toggle
* ✅ Persistent storage using LocalStorage
* ✅ Fully responsive design

---

## 🛠 Tech Stack

* **HTML5**
* **CSS3 (Glassmorphism UI + Animations)**
* **Vanilla JavaScript (ES6)**
* **LocalStorage API**
* **Font Awesome**
* **Canvas Confetti**

---

## ⚡ Performance Optimization

* Images resized and converted to **WebP**
* Reduced animation load for mobile
* Optimized slideshow transitions
* Lightweight DOM updates
