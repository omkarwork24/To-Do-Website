🚀 Task Manager Web App

A responsive To-Do / Task Manager application built using HTML, CSS, and JavaScript, focused on clean UI, efficient state management, and browser storage.

🔗 Live Demo:
https://omkarwork24.github.io/To-Do-Website/

💻 GitHub:
https://github.com/omkarwork24/To-Do-Website

📌 Features
Add, delete, and mark tasks as completed
Search and filter tasks (All / Active / Completed)
Progress tracking bar
Dark mode toggle
LocalStorage for data persistence
Responsive design (mobile, tablet, desktop)

🛠 Tech Stack
HTML5
CSS3
JavaScript (ES6)
LocalStorage

⚡ Key Highlights
Efficient DOM manipulation and event handling
State persistence using browser storage
Optimized UI with minimal re-renders
Performance-focused (optimized images & animations)
